## Installation
Ouvrir le fichier : `lib/app/config.php`

++++++++++++++++++++++++++++++++++++++++++++

#### Urls

`BASEADMIN -> l'url du site`

`BASEFRONT -> l'url de l'administration`

Pour les 2 urls ne pas oublier le "/" à la fin

#### Base de donnée

`DBNAME -> Le nom`

`HOST -> Le serveur`

`USER -> L'utilisateur`

`PASS -> Le mot de passe`

#### Cryptage

`KEYSHA1 -> la clé de salage pour le cryptage`
